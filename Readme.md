# Vero Links

Landing page minimalista desenvolvida para centralizar os principais canais da Vero Softwares.

O projeto foi criado para ser utilizado como link na bio do Instagram, direcionando visitantes para os canais mais importantes da empresa de forma simples, rápida e alinhada à identidade visual da marca.

## Objetivo

Transformar o tradicional "link da bio" em uma experiência mais profissional, oferecendo acesso rápido para:

* Solicitar Projeto
* Portfólio de Projetos
* Site Institucional
* Instagram

## Tecnologias

* HTML5
* CSS3
* JavaScript (ES Modules)
* Vercel (Deploy)

## Como Executar

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/vero-links.git
```

Acesse a pasta:

```bash
cd vero-links
```

Abra o arquivo `index.html` ou utilize uma extensão como Live Server.

## Deploy

O projeto está preparado para deploy na Vercel.

Após conectar o repositório ao GitHub, a publicação é realizada automaticamente a cada novo commit.
